  • [设备操作与维护] 调节Einstart – S平台教程

    zhaobin20160307@sina.com  0

    大家好，今天我给大家讲一下如何调节3D打印机平台。   首先，连接打印机至电脑。连接数据线后，长按”OK“键，打开打印机电源。   打开3D Start软件，点击“连接设备”，这时候会出现3D打印机的信息，显示已注册状态。   直接点击“连接”，连好打印机之后，点击“设置”。在上方任务框中，点击“平台调平”，选择“Z轴上升”，这时候，它就会一直上升到你设置好的Z轴高度。   如果觉得距离比较合适的话，可以选择对角线移动。接着，再选择另外一个对角线，看看平台和喷嘴的距离是否一致。   如果出现不一致的状况，我们可以对平台下方的三个螺丝进行调节。   拧紧螺丝，就可以拉开平台和喷嘴的距离。如果平台和喷嘴离的太远，那我们就可以选择把螺丝拧松一点，平台和喷嘴就会靠的比较近了。   调节完螺丝后，选择四角移动选项，这样你就可以看到喷嘴在整个平台上移动的距离是否一致。   到这里，我们就可以开始打印操作了。

  • [设备操作与维护] Einstart – S开箱及一键打印教程

    shuichiyiou@sina.com  0

    收到打印机后，进行拆箱。包装盒中共有五个物件：打印机、工具包、电源适配器、打印平台及打印材料。   1. 将打印机取出，解掉捆扎打印喷头的线扎。然后，取出用于保护内部运动结构的两块泡沫。接着，取出电源适配器，将电源适配器与打印机连接。   2. 从同一包装中取出导丝软管，将导丝软管插入打印机上端接口。取出打印材料，除去打印材料最前面的一截后，将其放入打印机料盘槽内。把材料通过打印机上端的孔位伸入至导丝软管。同时，将导丝软管的另一端插入到喷头的管子当中。   3. 打开电源，如机器显示蓝光，则说明已经接通电源。接着，长按OK键3秒，待蓝光变为白光，显示屏上出现“Welcome”字母，说明机器已经打开。   4. 下一步选择进丝，等待喷头的温度上升至195℃后，喷头部位的齿轮会将打印丝咬合进去。此时，说明进丝成功。整个过程大致需要2分钟左右。   5. 待喷嘴处有线丝挤出时，则说明打印丝已成功加载。下面，点击左键，停止进丝。在进丝完成后，就可以进行打印操作了。   6. 取出打印平台，放置于打印平台的支架上，平台下方的磁铁会与支架上的磁铁吸附在一起，保证平台的稳固。   7. 配套的TF卡内已经存有部分3D数据模型，大家可以点击右键，选择自己喜欢的数据进行打印。   从开箱到进行第一个打印作业的时间连10分钟都不到，相当便捷。

  • [3D打印基础知识]------生物3D打印技术

    3D创客营  0

    生物3D打印技术(Biological 3D Printing)       3D生物打印机基于现有技术发明，这些技术当前被用以制造工业零部件的3D模型。生物打印机的不同之处在于，它不是利用一层层的塑料，而是利用一层层的生物材料或者细胞构造块，去制造真正的活体组织。           器官移植可以拯救很多人体器官功能衰竭或损坏的患者生命，但这项技术也存在器官来源不足、排异反应难以避免等弊端。不过，随着未来“生物打印机”的问世，这些问题将迎刃而解。3D生物打印机可以有多个打印喷头，喷头可以打印人体细胞，被称为“生物墨”；也可以打印纯生物材料，被称为“生物纸”。所谓生物纸其实是主要成分是水凝胶，可用作细胞生长的支架。3D生物打印机使用来自患者自己身体的细胞，所以不会产生排异反应。         生物打印机首先“打印”器官或动脉的3D模型，接着将一层细胞置于另一层细胞之上。打印完一圈“生物墨”细胞以后，接着打印一张“生物纸”凝胶。不断重复这一过程，直至打印完成新器官。随后，自然生成的细胞开始重新组织、熔合，形成新的血管。每个血管大约需要一小时形成，而熔合在一起需要数天时间。           不过，从人体细胞、组织乃至器官被“打印”出来，到真正应用于临床，还有相当长一段路需要走。但是，随着生物打印技术的不断发展，最终有一天，只需轻轻按下按钮，就能让3D生物打印机制造出我们所需要的器官。

  • [3D打印基础知识]------金属粉末烧结成型技术

    3D创客营  0

    金属粉末烧结成型技术（Selective laser melting）       SLM成型过程原理与SLS基本相同。SLM技术使用材料多为不同金属组成的混合物，各成分在烧结过程中相互补偿，有利于保证制作精度。为了保证金属粉末材料的快速熔化，SLM技术需要高功率密度激光器，光斑聚焦到几十μm到几百μm。SLM技术目前最常使用光束模式优良的光纤激光器的激光功率在50w以上，功率密度达5×106W/cm2 以上。       SLM是极具发展前景的金属零件3D打印技术。SLM成型材料多为单一组分金属粉末，包括奥氏体不锈钢、镍基合金、钛基合金、钴-铬合金和贵重金属等。激光束快速熔化金属粉末并获得连续的熔道，可以直接获得几乎任意形状、具有完全冶金结合、高精度的近乎致密的金属零件。其应用范围已经扩展到航空航天、微电子、医疗、珠宝首饰等行业。SLM成型过程中的主要缺陷有球化、翘曲变形。 金属粉末烧结成型技术的优势： 1、成型精度和力学性能好。SLM是选择性激光熔化，顾名思义也就是在加工的过程中用激光使粉体完全熔化，不需要黏结剂，成型的精度和力学性能都比SLS要好。 2、不受形状和结构的限制，可以最大化地实现模具流道系统的合理化设计和布局，流道表面光洁顺滑，从而使注塑出来的产品质量更高，成型时间更短，生产废料更少。 3、用来烧结成型的金属材料品种多样，性能优异，可以满足不同生产领域的模具品质需求，如注塑模具、压铸模具、拉伸模具，无需渗铜工艺。 金属粉末烧结成型技术的劣势： 1、致密度仍需优化。SLM成型过程中，高能量密度的激光束将金属粉末瞬间熔化，然后又快速冷却，此过程非常不稳定，造成成型件表面和内部产生大量气孔导致成型件致密度不高。 2、悬垂结构尚在改进之中。悬垂结构的成型主要是预先添加支撑以保证成型过程的稳定性，成型结束后再去除支撑，获得完整的零件。悬垂结构使SLM成型零件的局部形状精度、尺寸精度不能达到要求，或者成型过程失败。

  • [3D打印基础知识]------多喷头打印技术

    3D创客营  0

    多喷头打印技术         使用光固化液体塑料在工作台上逐层打印薄层，并在打印过程中使用实蜡支撑打印材料，从而得到完整的模型。在每一层完成时，紫外线灯就会开始固化工作，然后工作台就会下降制作下一层。逐层堆砌的过程将会持续到整个模型打印完成。 （常见机型）       经其打印出来的产品质量高、寿命长，能够满足各种工程和机械领域内的应用，包括功能测试、成型装配校对、快速成型设计、设计通讯、以及快速加工等。该技术可以同时将柔性和刚性的混合材料打印、融合在一起。打印材料包括多种明暗效果的打印颜色，包括透明、不透明、黑色、白色、以及各种色差的灰色效果。     多喷头打印技术的优势： 1、打印成品质量高。一系列优质材料保证了高清晰度零件的生产，独特的蜡基支撑结构， 去除方便、快捷，即便是对于形状复杂、难以接触的内部几何结构而言。 2、支持多材料打印。粘土、橡皮泥、sugru、硅胶RTV，陶瓷，ABS，PLA，尼龙等等均在其可打印范围。 多喷头打印技术的劣势： 1、多喷头结构复杂。 2、维修难度高。 3、耗材被垄断，价格高。 4、打印速度慢。
